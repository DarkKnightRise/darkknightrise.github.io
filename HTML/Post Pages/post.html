How to use Termux-API

Published June 9 2019

This article is about how to call in termux by using termux api

Note:
This process required regular call charges

First of all install Termux-API from Play Store Now open termux and type below mention commands
$ pkg update upgrade
$ pkg install termux-api

Send SMS

To send sms in termux
$ termux-send-sms -n 123-------

Call

To Call in termux
$ termux-telephony-call 123-------
