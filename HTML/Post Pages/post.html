Termux-API

This article is about how to call in termux by using termux api

Note:

This process required regular call charges

First of all install termux api from Play Store
Termux-API

Now open termux and type below mention commands

$ pkg update upgrade
$ pkg install termux-api

Send SMS

To send sms type termux-api send-sms (number)

$ send-sms -n 123-------
